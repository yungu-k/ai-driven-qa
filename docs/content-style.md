# 콘텐츠 스타일 정책

- **문체**: 여정기(`repetition-to-ai-judgment-to-human`)만 존댓말(-습니다) — 독자에게 말 거는 종합 회고이기 때문. 나머지 개별 포스트는 평서체(-다)로 통일. 한 글 안에서 섞지 않는다.
- **표기**: "릴리스"(O) / "릴리즈"(X).
- **수치**: 실측값과 어림값을 구분해 표기한다("약 600건" vs "실측 615건"). 근거 없는 100%·전량류 표현 금지.
- **도구명**: Jira·Confluence·Figma·Google Sheets 등 도구 이름은 실명 표기(첫 등장 정식명, 이후 축약 가능). 사내 서버 주소·티켓 번호·고객사명·크리덴셜은 계속 비공개.
- **영어 용어**: 비개발 독자용 글에서는 첫 등장 시 우리말 병기 — 예: "월보드(벽에 거는 상황판)", "off-by-one(참조가 한 행씩 밀리는) 에러".
